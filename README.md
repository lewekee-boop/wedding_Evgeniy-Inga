# Свадебный лендинг Инги и Евгения

Одностраничный статический сайт-приглашение на свадьбу.

## Данные

- Дата: 06.08.2026
- Регистрация: 12:30
- Заезд в коттедж: 15:00
- Адрес: г. Тюмень, Целиноградская, 18

## Структура

- `index.html` — основная страница
- `assets/style.css` — стили сайта
- `assets/wedding-hero.png` — изображение первого экрана
- `assets/main.js` — счетчик до даты свадьбы
- `assets/favicon.svg` — иконка сайта
- `site.webmanifest` — manifest для браузеров
- `robots.txt` — разрешение индексации
- `.nojekyll` — отключение Jekyll на GitHub Pages

## Публикация на GitHub Pages

1. Загрузить все файлы в репозиторий.
2. В GitHub открыть `Settings` → `Pages`.
3. В `Build and deployment` выбрать `Deploy from a branch`.
4. Указать ветку `main` и папку `/root`.
5. Сохранить настройки и дождаться публикации.
